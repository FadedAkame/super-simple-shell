# ssSH
ssSH (super-simple-shell) is just a tiny concept UNIX shell that does the basic I/O, fork and executes processes.
It uses your existing environment variables by default.

# Building
> Tested on Linux with Clang & GCC.

```
chmod +x ./build.sh
./build.sh

./shell
```

## Note
by default, build.sh uses GCC. Change it to your prefered compiler.
